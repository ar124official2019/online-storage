Create New Directories

Allowed characters are 'a to z', 'A-Z', ., - and _
Directory name is required
Directories List
  • Create Directory option let you create one or more directories
  • To create a directory, use Directory Name input field
  • Click 'Add Next' to add typed directory.
  • Click 'Create Directories' to create directories in Directories list.
  • Selected directories are always there at bottom, so you may remove one if you wish to cancel.
