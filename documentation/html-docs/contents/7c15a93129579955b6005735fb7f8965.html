Attributes

Visibility Name Type Description
public email String

email is the email address of user whose file system is being delivered.

Operations

Visibility Name Description
public get()

get returns file system entries.

It accepts a query parameter, the path from root to access a particular file system entry or if not provided, it would return root directory contents.

{ path: String }
public post()

post method is used to create file system entries. This can be used upload files or create directories.

If a only path is sent, it would create a directory. If a file is sent(uploaded), it would save file to given path.

Body Format: { path: Strnig, file: File }
public delete()

delete method deletes the file system entry pointed by path.

Body Format: { path: String; }
public put()

put method renames a file system entry point by path in body to targetPath in body. If special flag keep is true, it would just copy file to target and keeps original.

Body Format: { path: String, targetPath: String, keep: Boolean }
